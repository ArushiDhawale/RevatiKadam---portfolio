# Revati Kadam Portfolio

A personal portfolio website showcasing my skills, projects, and background. Built using HTML, CSS, and JavaScript, it features a responsive design, dynamic theme toggling, and interactive elements to provide an engaging user experience.

## Features

- **Interactive UI**: Smooth navigation between sections (Home, About, Skills, Projects, Contact).
- **Dynamic Theming**: Light and dark mode support, along with customizable accent colors via a settings menu.
- **Typing Animation**: Engaging typing text effect on the Home section.
- **Contact Form**: Functional contact form integrated with EmailJS.
- **Responsive Layout**: Designed to work seamlessly across different devices with a sidebar navigation.
- **Downloadable Resume**: Direct link for users to download my latest resume.

## Technologies Used

- **HTML5**: For structuring the content.
- **CSS3**: For styling, layout, and animations (including custom properties for theming).
- **JavaScript (Vanilla)**: For interactivity, DOM manipulation, and logic handling.
- **EmailJS**: For sending emails directly from the contact form without a backend server.

## Installation & Setup

To view the project locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/revatikadam0607/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Open `revati-kadam-portfolio/index.html` in your preferred web browser.

Alternatively, you can use a local development server like Live Server in VS Code for a better development experience.

## Project Structure

- `revati-kadam-portfolio/`: The main folder containing the portfolio application.
  - `index.html`: The main entry point.
  - `css/`: Contains styling files (`style.css`, `themes.css`, etc.).
  - `js/`: Contains logic scripts (`script.js`, `theme.js`, etc.).
  - `assets/`: Stores images, resumes, and other static assets.
  - `data/`: Contains data-related files if any.
